# The Rulebook

We are committed to providing a welcoming and inclusive environment for all contributors and users of the waytosea project. This project is a simple repository hosting themes and wallpapers for everyone to enjoy.

## Expected Behavior

- **Be respectful** and professional in all interactions
- Welcome new contributors and help them get started
- Provide constructive feedback on themes and wallpapers
- Respect others' creative work and intellectual property
- Report issues and suggestions in a courteous manner
- **always remember to credit the original artists**

## Unacceptable Behavior

- Harassment, discrimination, or disrespect toward any individual
- **Offensive language or inappropriate content**
- Spam or malicious activity
- Copyright or intellectual property violations
- Any form of _abusive behavior_

## Enforcement

Violations of this Code of Conduct may result in temporary or permanent removal from the project.

> Thank you for helping keep waytosea a positive and creative community!
