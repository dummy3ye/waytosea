# Installation

To get started with **Way To Sea**, follow these steps:

## Prerequisites

- [Git](https://git-scm.com/) installed on your machine.

## Clone the Repository

1. Open your terminal or command prompt.
2. Clone the repository:

   ```bash
   git clone https://github.com/dummy3ye/waytosea.git
   ```

3. Navigate into the project directory:

   ```bash
   cd waytosea
   ```

## Setup

The project is a static website. You don't need to install any server-side dependencies to view it locally:

1. Open `website/index.html` in your preferred web browser.
2. Alternatively, you can use a live server extension (like "Live Server" for VS Code) to preview changes in real-time.
